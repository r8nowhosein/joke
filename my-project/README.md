# My Project

Detta projekt hämtar skämt från ett api där man kan spara sina skämt och hämta upp nya skämt

## Klona 
https://github.com/r8nowhosein/joke.git

## Beskrivning
En enkel projektmall för att komma igång snabbt.

## Installation
se till att först vara i root filen "my-project"
npm install


## Användning
npm run dev

## öppna localhost
  http://localhost:5173/
